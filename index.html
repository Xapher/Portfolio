About the game:

Without giving too much away, it will be an environment building game that develops as you brew coffee as a barista. It will have dynamic weather, and modifiable large width terrains. Possibly even dynamic floating terrain.

I'm currently working on the terrain generation part of the game as that will be the biggest part. As well as I have already made a start to the brewing mechanics. Which will be going into more complicated coffee systems like blooming, coffee strength, roast type, flavor profile, etc.

Progress: Working on it when not working on other projects.

About the Terrain Generator:

A Planned unity plugin that will allow you to create massive terrain that looks good, as realistic as possible using different types of erosions and weathering systems.

I'm currently re-writing the generator that I had before. It was using C# script using the cpu, which started out fine. However, adding more developments to it severely degraded the generating time. So I'm currently converting it to use compute shaders to take advantage of mass parallelization. The CPU Generator was able to run pretty good erosion and I was starting to work on a biome system based on the erosion based off of humidity from the water.

Progress: re-wrote to generate height and currently working on realistic erosion.
